# Chicago-crime
The project aims to analyze large-scale crime data from the city of Chicago to identify recurring patterns and potential serial crime behaviors based on time, location, and crime type. Spark RDD and Spark SQL will be used for analysis, with optional clustering techniques for pattern discovery.
Dataset:
Chicago Crime Dataset (2001–Present)
https://data.cityofchicago.org/Public-Safety/Crimes-2001-to-Present/v9q9-3dm2
